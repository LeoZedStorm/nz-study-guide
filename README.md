# NZ Study Guide · 新西兰留学指南

[![Deploy](https://github.com/LeoZedStorm/nz-study-guide/actions/workflows/pages-deploy.yml/badge.svg)](https://github.com/LeoZedStorm/nz-study-guide/actions/workflows/pages-deploy.yml)
[![GitHub stars](https://img.shields.io/github/stars/LeoZedStorm/nz-study-guide?style=social)](https://github.com/LeoZedStorm/nz-study-guide)

**👉 https://leozedstorm.github.io/nz-study-guide/**

一个新西兰留学生活实用指南：签证、住宿、驾照、打工、医疗这些内容对所有新西兰留学生通用，不限于某个学校或项目。

同时，本站也是**四川大学-奥克兰大学计算机科学与技术（中外合作办学）**项目的专属介绍站，整理了培养计划、课程模块，以及奥克兰大学（UOA）相关的情况。

## 内容包括

- 🧭 签证、住宿、语言、选课、打工、医疗、驾车等留学生活攻略（按"行前 → 落地 → 校园日常"分组）
- 🎓 SCU-UOA 项目的培养计划、课程说明（含外方课程官方链接）
- ❓ 项目常见问题 FAQ、学习资源导航
- ✈️ 奥克兰及新西兰旅行小贴士

<summary>本地开发</summary>

基于 [Jekyll](https://jekyllrb.com/) + [Chirpy](https://github.com/cotes2020/jekyll-theme-chirpy) 主题，通过 GitHub Actions 自动构建部署到 GitHub Pages，推送到 `main` 分支即自动上线。

本地预览需要 Ruby ≥ 2.7：

```bash
bundle install
bundle exec jekyll serve
```

浏览器打开 `http://127.0.0.1:4000` 预览。
